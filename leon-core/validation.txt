Leon Validation
===============

Requrirements
-------------
* Server- and client-side validation
* central declaration of validation rules (client- and server-side)
* build-in validation rules
* user-definied/ custom validation rules
* integration for angular.js / bootstrap

Build-in validation rules
-----------------------
* Numeric validator
* Number range validator
* Not Empty validator
* Min-/Max-length validator
* Date validator
* Email validator
* Regex validator



Valdation response
------------------
HTTP Status: 422 ("Unprocessable Entity")
Reponse Body:
{
	errors: [
		{ 
			field: "name",
			message: "Field is requried"
		},
		...
	]
}
